## To run this project do the following
NOTE: This is a pre-build app not using Expo Go, so please follow the below and pre-build in step 2.

1. Install dependencies

   ```bash
   yarn install
   ```

2. Pre-build the app

   ```bash
   yarn prebuild
   ```

3. Start the app (ios or android)

   ```bash
   yarn ios
   ```

   ```bash
   yarn android
   ```
